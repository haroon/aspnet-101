## Installation (Arch)

### .Net runtime

* yay -S dotnet-host-bin
* yay -S dotnet-runtime-bin
* yay -S dotnet-targeting-pack-bin
* yay -S dotnet-sdk-bin

### ASP.NET Framework

* yay -S aspnet-targeting-pack-bin
* yay -S aspnet-runtime-bin 

## Running

* cd server
* dotnet run

## Verifying
* go to http://localhost:5199/plugin/getall
* verify output
